## Technical Architecture

### Trace Flow

```
API Request (FastAPI)
  ├─ HTTP Span (auto-instrumented)
  ├─ Database Span (auto-instrumented)
  └─ Workflow Trigger (manual span)
     └─ Orchestrator Workflow
        ├─ Activity: Context Build
        ├─ Activity: Dispatch Agent
        │  └─ NATS Publish (context propagated)
        │     └─ Agent Worker (receives and traces)
        │        ├─ Agent Execution Span
        │        └─ LLM Call Span
        │           └─ Result with Token Info
        └─ Activity: Gate Approval
```

### Service Instrumentation

| Service | Method | Key Spans |
|---------|--------|-----------|
| mc-backend | FastAPI Auto | HTTP methods, DB queries, business logic |
| orchestrator | Manual + Import | Workflows, activities, gates |
| agent-* | Manual | NATS handling, LLM calls, agent logic |
| otel-collector | N/A | Infrastructure component |
| jaeger | N/A | Trace storage & indexing |

### Export Paths

```
Services
  ├─ mc-backend (FastAPI) → OTEL SDK
  ├─ orchestrator (Temporal) → OTEL SDK
  ├─ agent-workers (Base) → OTEL SDK
  └─ otel-collector (Port 4317/4318)
     ├─ otlp/jaeger (gRPC to jaeger:14250)
     ├─ jaeger/http (HTTP to jaeger:14268)
     ├─ metrics → prometheus:9090
     └─ logs → loki:3100
```

## Deployment Checklist

To deploy this implementation:

1. **Install Dependencies**
   ```bash
   pip install -r repos/infra/observability/requirements.txt
   ```

2. **Start Infrastructure**
   ```bash
   cd repos/infra
   docker-compose up -d
   ```

3. **Verify Services**
   ```bash
   docker ps | grep -E "jaeger|otel|postgres|nats"
   ```

4. **Start Services**
   - Orchestrator: `cd repos/orchestrator && ENVIRONMENT=dev python worker.py`
   - Backend: `cd repos/mc-backend && ENVIRONMENT=dev python main.py`
   - Agents: Agent-specific launchers

5. **Access Jaeger UI**
   - Open http://localhost:16686 in browser

6. **Generate Test Traces**
   - Make API requests to trigger end-to-end traces

7. **Verify Traces**
   - Check Jaeger UI for services, operations, and spans
