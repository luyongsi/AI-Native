================================================================================
SEPARATION OF CONCERNS - CRITICAL POINT
================================================================================

Location: a9/a9_dev_agent.py, lines 80-85

Implementation:
  diff_for_audit = {
      "files_changed": final_diff.get("files_changed", []),
      "changes_summary": final_diff.get("changes_summary", "")
  }

What Coder Produces (Hidden):
  - self_inspection: reasoning, confidence, issues
  - metadata: files_created, worktree_path, etc.

What Auditor Receives (Only):
  - files_changed: file list with path, type, language
  - changes_summary: brief description

Result:
  - Auditor makes independent decision
  - Cannot be influenced by Coder assessment
  - Enables adversarial code review
