================================================================================
INTEGRATION POINTS
================================================================================

With A5 Design Review:
  - Triggered after review.completed
  - Uses spec from review payload

With Orchestrator:
  - Publishes architecture.dag_built NATS event
  - Consumed by Dev Agent scheduler
  - Drives task execution sequencing

With Dev Agents:
  - Receive DAG with tasks and critical path
  - Execute in topological order
  - Report progress to task_executions table

With Database:
  - Persists to task_dags, task_executions, shared_modules
  - Indexed by req_id for fast lookups
  - JSONB for flexible DAG storage
