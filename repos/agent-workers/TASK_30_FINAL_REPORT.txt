================================================================================
TECHNICAL ARCHITECTURE
================================================================================

5-PHASE EXECUTION PIPELINE:

Phase 1: Semantic Search (100-500ms or 10ms fallback)
  - Calls /api/knowledge/search
  - Returns top-5 similar requirements
  - Fallback to static KB if API unavailable

Phase 2: Dependency Query (100-300ms or 0ms fallback)
  - Neo4j: MATCH (req)-[:DEPENDS_ON]->(service)
  - Returns service dependencies
  - Skips if NEO4J_URL not configured

Phase 3: Related PRs Query (10ms placeholder)
  - Query PostgreSQL for related changes
  - Framework ready for GitHub/GitLab integration
  - Currently returns empty list

Phase 4: Knowledge Fusion (50-100ms)
  - Extract code patterns from metadata
  - Identify risks from tags
  - Calculate complexity score
  - Estimate 3-20 day range
  - Call LLM for intelligent summary

Phase 5: Event Publishing (5ms)
  - Publish artifact.produced.A2 event
  - Push to NATS JetStream
  - Payload: full knowledge package

TOTAL EXECUTION TIME: 200ms - 2s (depends on API availability)
