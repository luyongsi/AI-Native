2. API SCHEMA GENERATOR MODULE
   File: a4/api_schema_generator.py
   Size: 21 KB
   Lines: 560
   Status: NEW

   Features:
   - Generates OpenAPI 3.1 specifications from requirement text
   - LLM-powered generation using DeepSeek API
   - Few-shot prompting with 2 built-in examples
   - Schema validation with automatic retry (max 3 attempts)
   - Response parsing (handles markdown, JSON wrapping)
   - Graceful fallback on LLM failure
   - Comprehensive logging and validation tracking

   Key Classes:
   - APISchemaGenerator
     * async generate(requirement_text, context, max_retries=3) -> Dict
     * _build_prompt(requirement_text, title, domain) -> str
     * _format_fewshot_examples() -> str
     * async _call_llm(prompt) -> Optional[str]
     * _parse_llm_response(response) -> Tuple[Dict, Optional[str]]
     * async _validate_and_retry(...) -> Tuple[bool, Dict, List[str]]
     * _generate_fallback(title, domain) -> Dict

   Few-Shot Examples:
   - Example 1: User Authentication API
     * Login endpoint (/auth/login)
     * User profile endpoint (/users/{id})
     * User schema with UUID, email, name, timestamps
   - Example 2: E-Commerce API
     * Product listing (/products)
     * Order creation (/orders)
     * Product, Order, OrderCreate, OrderItem schemas
