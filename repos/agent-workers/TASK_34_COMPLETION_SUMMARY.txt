ITERATION FLOW
================================================================================

Max 3 iterations:

Iteration 1:
  Coder gen → Auditor review
  ├─ Approved? → Done
  └─ Rejected? → Iteration 2

Iteration 2:
  Coder gen (w/ feedback) → Auditor review
  ├─ Approved? → Done
  └─ Rejected? → Iteration 3

Iteration 3:
  Coder gen (w/ feedback) → Auditor review
  ├─ Approved? → Done
  └─ Rejected? → Escalate (human)
