================================================================================
IMPLEMENTATION OVERVIEW
================================================================================

A circuit breaker system that automatically escalates agent execution through
three levels of increasing intervention:

  Level 1: FEW_SHOT (1st failure)
    - Injects task-specific examples to improve agent guidance
    - Uses original model (DeepSeek v3)

  Level 2: STRONG_MODEL (2nd failure)
    - Switches to Claude 3.5 Sonnet (higher reasoning capability)
    - Lower temperature for deterministic behavior

  Level 3: HUMAN (3rd failure)
    - Sends Feishu webhook notification to operations team
    - Attempts final time with Claude Opus (highest quality)
    - After this, escalation exhausted and task fails

Successful execution at any level resets the circuit breaker to NORMAL.
