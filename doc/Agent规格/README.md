## 📐 规格模板结构

每个 Agent 规格包含：

1. **基本信息**: Agent ID, Type, 订阅/发布事件, 代码位置
2. **职责**: 3-5 句话核心描述
3. **输入**: Context Package 结构 + 关键字段
4. **处理流程**: Phase 1/2/3... 的详细步骤
5. **输出**: 返回结构 + 持久化位置
6. **LLM 调用**: 任务类型、温度、Prompt 结构
7. **依赖**: 上游/下游/外部服务
8. **当前实现状态**: ✅已实现 / ⚠️部分实现 / ❌未实现
9. **已知问题**: 问题 + 影响 + 建议
10. **测试方法**: 单元测试 / NATS 触发 / 端到端测试

参考: `A1-需求分析Agent完整设计.md`
