NATS EVENTS
===========

test.validate (API -> A7)
- Triggers validation of a test case

test.validated (A7 -> Backend/Frontend)
- Returns validation results

test.augment_request (API -> A11)
- Requests test augmentation

test.augmented (A11 -> Backend/Frontend)
- Returns augmentation results

Validation Flow:

-> A11 processes -> A11 publishes test.augmented
-> Frontend refreshes with new tests
